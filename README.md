# Chessinator-9000
A smart open source folding chessboard. Future expansions planned with electromagnetic gantry.

# About
This board utilizes a network of hall sensors to track and log moves. Made up of 2 adjacent PCBs and designed to be foldable for improved portability. Future plans include a supplementary electromagnetic gantry and API connection to Lichess and Chess.com.
