# Chessinator-9000
A smart open source folding chessboard. Future expansions planned with electromagnetic gantry.
<img width="866" height="436" alt="image" src="https://github.com/user-attachments/assets/b57582f4-b575-4bf7-bf63-68f10355a42e" />

# About
This board utilizes a network of hall sensors to track and log moves. Made up of 2 adjacent PCBs and designed to be foldable for improved portability. Future plans include a supplementary electromagnetic gantry and API connection to Lichess and Chess.com. Currently configured to project move and take a crowdvote for Open Sauce.
